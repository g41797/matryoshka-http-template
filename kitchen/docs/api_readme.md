# matryoshka-http-template

Template repository for building HTTP services using matryoshka.

---

## Features

- **Odin + matryoshka:** High-performance, low-latency pipeline architecture.
- **HTTP/1.1:** Via odin-http and an handlers.
- **Unified Master Model:** Clean separation between stage logic and infrastructure.
- **Pre-configured:** VS Code debugging, CI/CD, and documentation generation.
